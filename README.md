# Forma Habit Tracker
FOIM a Forma is a simple and intuitive habit tracker app built with Swift and SwiftUI. This app helps users manage and track their daily habits easily.

Features
Add Habits: Users can add multiple habits they want to follow every day.
Edit & Delete: Users can edit or delete any habit at any time.
Home Screen Overview: All habits are displayed on the home screen for easy tracking.
Daily Streak: Users earn a streak value when they complete all their habits for the day. The streak updates daily to motivate consistent habit building.
